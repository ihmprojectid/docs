</Info>

## Jalankan preview lokal

<Steps>
  <Step title="Install the Mintlify CLI">

```bash
npm i -g mint
```
  </Step>
  <Step title="Preview locally">

```bash
mint dev
```

Preview lokal biasanya tersedia di `http://localhost:3000`.
  </Step>
</Steps>

## Struktur konten yang dipakai

<AccordionGroup>
  <Accordion title="Guide">
    Berisi onboarding, panduan operasional, best practice, dan troubleshooting produk.
  </Accordion>
  <Accordion title="API Reference">

## Prinsip penulisan

<Checklist>
  <Check>Gunakan bahasa Indonesia yang natural dan langsung ke use case</Check>
  <Check>Prioritaskan kejelasan alur, contoh payload, dan outcome user</Check>
  <Check>Jangan ubah gaya visual brand; cukup rapikan struktur, copy, dan navigasi</Check>
  <Check>Jika fitur masih berkembang, tandai statusnya dengan jujur</Check>
</Checklist>

## Validasi sebelum commit

```bash
mint broken-links
```

## Jika port bentrok

```bash
mint dev --port 3333
```

## Troubleshooting authoring

<AccordionGroup>
  <Accordion title="Preview tidak jalan">
    Pastikan Node.js sudah sesuai dan `mint` terpasang global. Jika perlu, reinstall dengan `npm i -g mint`.
  </Accordion>
  <Accordion title="Link internal rusak">
    Jalankan `mint broken-links`, lalu cek slug halaman di `docs.json` dan frontmatter tiap file MDX.
  </Accordion>
</AccordionGroup>
