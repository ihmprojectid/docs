Panduan ini menjelaskan langkah-langkah untuk mengimpor produk digital dari platform **Scalev** ke [**Daekan.app**](http://Daekan.app) menggunakan **Business API Key**.

## 1. Buka Menu Digital Download di Daekan

  <img

    alt="Screenshot From 2026 06 15 13 49 53"
    title="Screenshot From 2026 06 15 13 49 53"
  />
</Frame>

Masuk ke dashboard [**Daekan.app**](http://Daekan.app), lalu buka menu:

<Frame>
  <img
    src="/images/Screenshot-from-2026-06-15-13-50-23.png"
    alt="Screenshot From 2026 06 15 13 50 23"
    title="Screenshot From 2026 06 15 13 50 23"
  />
</Frame>

**Products → Digital Download**

Setelah itu, klik tombol **Import Produk**.

---

## 2. Pilih Platform Scalev

<Frame>
  ![Screenshot From 2026 06 15 13 50 47](/images/Screenshot-from-2026-06-15-13-50-47.png)
</Frame>

Pada popup **Import Produk**, pilih platform **Scalev**.

Scalev digunakan untuk mengimpor produk dari akun Scalev ke Daekan menggunakan API Key.

Klik **Lanjut** untuk melanjutkan proses.

---

## 3. Masukkan API Key Scalev

<Frame>
  ![Screenshot From 2026 06 15 13 51 13](/images/Screenshot-from-2026-06-15-13-51-13.png)
</Frame>

Daekan akan meminta **Business API Key** dari akun Scalev.

API Key ini diperlukan agar Daekan dapat mengambil daftar produk dari Scalev.

> Catatan: API Key hanya digunakan untuk sesi import dan tidak disimpan di server Daekan.

---

# Cara Mendapatkan API Key dari Scalev

## 4. Buka Settings di Dashboard Scalev

<Frame>
  ![Screenshot From 2026 06 15 13 52 13](/images/Screenshot-from-2026-06-15-13-52-13.png)
</Frame>

Masuk ke dashboard **Scalev**, lalu buka menu:

**Settings**

---

## 5. Pilih Menu Developers

<Frame>
  ![Screenshot From 2026 06 15 13 52 36](/images/Screenshot-from-2026-06-15-13-52-36.png)
</Frame>

Di halaman Settings, pilih menu:

**Developers**

---

## 6. Buka Tab API Keys

<Frame>
  ![Screenshot From 2026 06 15 13 52 55](/images/Screenshot-from-2026-06-15-13-52-55.png)
</Frame>

Pada halaman Developers, pilih tab:

**API Keys**

<Frame>
  ![Screenshot From 2026 06 15 13 53 38](/images/Screenshot-from-2026-06-15-13-53-38.png)
</Frame>

Kemudian klik tombol:

**\+ Create API Key**

---

## 7. Buat API Key Baru

<Frame>
  ![Screenshot From 2026 06 15 13 54 48](/images/Screenshot-from-2026-06-15-13-54-48.png)
</Frame>

Isi data API Key sebagai berikut:

**API Key Name:**\
Contoh: `daekan`

**Description:**\
Contoh: `daekan`

**Key Type:**\
Pilih `Secret`

Setelah semua data terisi, klik tombol **Create**.

---

## 8. Salin API Key

<Frame>
  ![Screenshot From 2026 06 15 13 56 28](/images/Screenshot-from-2026-06-15-13-56-28.png)
</Frame>

Setelah API Key berhasil dibuat, Scalev akan menampilkan API Key.

Klik **Copy** untuk menyalin API Key tersebut.

> Simpan API Key dengan aman. Biasanya API Key hanya dapat dilihat satu kali setelah dibuat.

---

# Kembali ke Daekan

## 9. Masukkan API Key ke Daekan

<Frame>
  ![Screenshot From 2026 06 15 13 51 13](/images/Screenshot-from-2026-06-15-13-51-13.png)
</Frame>

Kembali ke halaman import produk di [**Daekan.app**](http://Daekan.app).

Tempel API Key Scalev pada kolom **Business API Key**, lalu klik:

**Ambil Daftar Produk**

---

## 10. Pilih Produk yang Ingin Diimport

<Frame>
  ![Screenshot From 2026 06 15 13 57 05](/images/Screenshot-from-2026-06-15-13-57-05.png)
</Frame>

Setelah daftar produk muncul, pilih produk yang ingin diimport.

Kamu bisa memilih produk satu per satu, atau klik **Pilih semua** untuk mengimpor semua produk.

Setelah memilih produk, klik tombol:

**Import Produk**

---

## 11. Import Berhasil

<Frame>
  ![Screenshot From 2026 06 15 13 58 05](/images/Screenshot-from-2026-06-15-13-58-05.png)
</Frame>

Jika proses berhasil, Daekan akan menampilkan halaman **Import Berhasil**.

Produk dari Scalev sudah berhasil diimport ke Daekan dan bisa langsung dikelola melalui menu produk Daekan.

Klik **Lihat Produk** untuk membuka daftar produk yang sudah diimport.

---

## Catatan Penting

Pastikan API Key Scalev memiliki akses atau scope berikut:

**product:list** dan **product:read**

Gunakan API Key dengan hati-hati dan jangan membagikannya kepada pihak lain.
